# Gnotes
Gnotes is a Front end React Project next.js based, for dealing with Scores mangment.

## Installation

Use the package manager yarn to install Gnotes.

```bash
git clone https://github.com/IGZO/Gnotes.git
```

## Usage

```
cd Gnotes
yarn install
yarn next dev
```
